# C++

Essentials:

Variables:

```c++
int a, b, sum;
cin >> a;
cin >> b;
sum = a + b;
cout << sum << endl;
```

Compile command (on linux):

``` bash
g++ input_file.cpp -o output_executable
```

` // Line comment `


Name spaces:

` using namespace std; `

Allows us to write ` cout << "string" ` instead of ` std::cout << "string" `
