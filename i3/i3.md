# i3 . Tiling manager

## Commands

To exit / logout : `Mod+shift+e`

## Installation / Customisation

On hig res screen text showed too small 

create `~/.Xresources`

```
! Fonts {{{
Xft.antialias: true
Xft.hinting:   true
Xft.rgba:      rgb
Xft.hintstyle: hintfull
Xft.dpi:       220
! }}}
```

then load using

`xrdb -load ~/.Xresources`
